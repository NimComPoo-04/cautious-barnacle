# cautious-barnacle

Simple 'the one who shall not be named' clone.

## Building and Running Steps

For Linux
```bash
gcc -Wall -Wextra -ggdb bothoristis.c -lX11
./a.out
```

For Windows (mingw64)
```bat
gcc -Wall -Wextra -ggdb bothoristis.c -luser32 -lgdi32 -o game.exe
game
```
